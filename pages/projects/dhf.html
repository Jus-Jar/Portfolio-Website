Project Overview

The Dynamic Highlight Function (DHF) project focuses on enhancing the Brightstart Reading Tutor by integrating an adaptive text-highlighting tool synchronized with synthetic speech. This tool addresses the challenges of early literacy development in linguistically diverse environments like the Caribbean by accommodating accents, dialects, and personalized reading paces. Built using React.js, Bootstrap, Python (Flask API), and MongoDB, the DHF creates tailored reading lessons that replicate the one-on-one experience of literacy specialists. The project demonstrates the potential to transform digital literacy tools, making them more inclusive and effective for young learners.

Key Features

  • 1) Adaptive Text Highlighting: Synchronizes text with synthetic speech, adjusting to individual reading paces and pronunciation styles.

  • 2) User-Friendly Interface:Built with React.js and Bootstrap for an intuitive online reading experience.

  • 3) Speech-to-Text Integration: Real-time synchronization of spoken words with highlighted text for visual and auditory learning.

  • 4) Personalized Reading Experience:Customizable reading pace and voice preferences (e.g., male voice, highlight color).

  • 5) High Accuracy: Over 95% word detection accuracy at fast and medium reading speeds.

Technology Stack

The Dynamic Highlight Function (DHF) leverages a modern technology stack to provide an effective, adaptive reading tool. The front-end is built using React.js and Bootstrap, ensuring a responsive and user-friendly interface for young learners. The back-end is powered by Python with the Flask framework, enabling efficient processing of reading materials and synchronization of text with synthetic speech. MongoDB is used for secure, scalable data storage, managing user preferences and reading lesson data. Additionally, a speech-to-text API is integrated to enable real-time word detection and highlight synchronization, offering a seamless, interactive reading experience. This stack ensures a reliable and flexible system that can easily adapt to user needs and improve reading engagement.
